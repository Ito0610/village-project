福祉事業所（移動支援・居宅支援）向けの、利用者様情報を整理・共有するためのシステムの要件・設計・実装メモです。

| ファイル | 内容 |
|----------|------|
| [01_要件定義書.md](./docs/01_要件定義書.md) | 背景・目的、機能要件、確定事項（Notion・規模・必須項目・AI） |
| [02_データモデル案.md](./docs/02_データモデル案.md) | 利用者・支援パターン・写真のエンティティ案 |

| [04_Notion設計.md](./docs/04_Notion設計.md) | 利用者DBのプロパティ、詳細ページテンプレート、支援パターン（子ページ） |
| [05_AIテキスト振り分け設計.md](./docs/05_AIテキスト振り分け設計.md) | テキスト→項目振り分けの流れ、項目一覧、プロンプト例、実装選択肢 |

## モック（確認・相談用）

`mock/` フォルダに、次のHTMLを用意しています。ブラウザで開いて画面の流れを確認し、修正点を相談できます。

- **利用者一覧.html** … Webアプリの「利用者情報一覧」のモック。各行から「Notionで開く」で1ページへ
- **1利用者1ページ.html** … 1利用者につき1ページに全情報を載せるNotionレイアウトのモック

詳細は [mock/README.md](./mock/README.md) を参照。
