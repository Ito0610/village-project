# モック（確認・相談用）

利用者情報整理システムの**画面イメージ**を確認するためのHTMLです。  
見ながら「ここを変えたい」「項目の順番を変えたい」などの修正点を相談できます。

## 使い方

1. **利用者一覧.html** をブラウザで開く（ダブルクリックまたはドラッグ＆ドロップでOK）
2. **情報一覧**として「利用者情報」と「施設情報」の2セクションが表示される
3. 検索欄にひらがな（例：やまだ、ほうかご）を入力すると、よみがな・読み仮名で絞り込まれる
4. 利用者行の「詳細を開く」をクリック → **1利用者1ページ.html** が開く
5. 1ページ内に「基本情報〜支援の流れ（パターン別）」まで並んでいることを確認

## 想定している流れ

- **情報一覧** = Webアプリで表示。利用者情報＋施設情報を一覧にし、ひらがなで検索可能
- **利用者** = 「詳細を開く」でその利用者さんのNotionページ（1ページに全情報）へ
- **1ページ** = 選んだ利用者さんのNotionページ。子ページや別画面への遷移はしない

## ファイル

| ファイル | 内容 |
|----------|------|
| 利用者一覧.html | 「情報一覧」のモック（利用者情報・施設情報・ひらがな検索） |
| 1利用者1ページ.html | Notionの「1利用者1ページ」のレイアウトモック（全セクションを1ページに配置） |
| **designs/** | **デザイン比較用**（福祉事業所向け・明るめの5パターン） |

### デザイン5パターン（designs フォルダ内）

福祉事業所向けの**明るめ**デザインを5種類用意しています。**designs/デザイン一覧.html** を開くと一覧で選べます。

| パターン | 特徴 |
|----------|------|
| **A：やわらかパステル** | うすいピンク・ブルー・ミントのグラデーション。角丸多めでやわらかい印象 |
| **B：清潔ライトブルー** | 水色ベースの清潔感。すっきり読みやすい |
| **C：自然グリーン** | クリーム＋やわらかい緑。自然で安心感のある明るさ |
| **D：あたたかオレンジ** | ベージュ＋コーラル・オレンジ。あたたかく迎え入れる印象 |
| **E：すっきりイエロー** | うすいイエロー＋ゴールド系。日差しのような明るさ |

**本番は 利用者一覧.html（現状の案）で進める想定です。** designs/ の5パターンは参考用として残しています。デザインや項目の並び・文言の修正希望があれば、相談時に伝えてもらえれば設計書やモックを更新します。
