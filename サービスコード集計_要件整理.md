# サービスコード集計マクロ 要件整理

## 1. マクロの概要

| 項目 | 内容 |
|------|------|
| マクロ名 | `サービスコード集計` |
| 目的 | 「様」付きシートのサービスコードを E5（受給者番号）単位で合算し、集計結果を出力する |
| 制約 | Mac対応・Dictionary 不使用（Collection と配列のみ使用） |

---

## 2. 対象シート

- **条件**: シート名に **「様」** を含むシートのみ対象
- **例**: `山田太郎様`、`佐藤花子様` など
- 「様」を含まないシートは処理対象外（スキップ）

---

## 3. 同一人物の判定

- **キー**: 各シートの **E5** の値（受給者番号として扱う）
- **ルール**: E5 が同じ値のシートは「同一人物」として一括して集計する
- 複数シートが同じ E5 を持つ場合、それらを 1 グループとして扱う

---

## 4. 入力データの取り込み範囲

- **範囲**: 各行 **16 行目～35 行目**
- **ブロック1**: 列 **R, S, T**（R=コード, S=単位数, T=内容）
- **ブロック2**: 列 **U, V, W**（U=コード, V=単位数, W=内容）
- コード（R列またはU列）が空でない行のみ集計対象

---

## 5. 処理①：サービスコード集計の出力

### 出力先

- 各 E5 グループ内で、**シート Index が最小のシート**（「先頭」の「〇〇様」シート）に出力

### 出力位置・内容

| 項目 | 内容 |
|------|------|
| 開始行 | **45 行目** |
| クリア範囲 | 45 行目～1000 行目（R～U 列） |
| 出力列 | **R**＝サービスコード, **S**＝内容, **T**＝単位数, **U**＝回数 |

### 集計ルール

- 同じサービスコードは **合算**（内容・単位数は先に出てきたものを採用、回数は加算）
- 出力時のコード順は **数値として昇順** にソート

---

## 6. 処理②：集計シートへの単位数合計の出力

### 前提

- シート名 **「集計」** が存在する場合のみ実行
- 存在しない場合はこの処理は行わない（エラーにはしない）

### 集計ロジック

1. 各 E5 グループに属する「〇〇様」シートについて、**V44** の値を合計する
2. V44 が空または数値でないセルは 0 として扱う
3. シート「集計」の **A 列**（5 行目以降）を上から検索し、**受給者番号（E5 の値）と一致する行**を探す
4. 一致した行の **K 列** に、上記で求めた **V44 の合計値** を書き込む

### 検索・書き込みの範囲

| 項目 | 値 |
|------|-----|
| 検索開始行 | 5 行目 |
| 検索終了行 | 1000 行目 |
| 照合列 | A 列（受給者番号） |
| 出力列 | K 列（単位数合計） |

### 注意

- A 列が空の行が現れた時点で、それ以降の検索は打ち切る（連続データを想定）

---

## 7. 定数一覧

| 定数名 | 値 | 意味 |
|--------|-----|------|
| `OUTPUT_START_ROW` | 45 | 集計結果の出力開始行 |
| `CLEAR_END_ROW` | 1000 | クリアする最大行 |
| `SUMMARY_SHEET_NAME` | "集計" | 単位数合計を書き込むシート名 |
| `SUMMARY_START_ROW` | 5 | 集計シートの検索開始行 |
| `SUMMARY_END_ROW` | 1000 | 集計シートの検索・書き込み上限行 |

---

## 8. 処理フロー概要

```
1. 「様」を含むシートを走査
   → E5 の値ごとにシートをグループ化

2. 各グループについて
   → R16:T35 と U16:W35 からサービスコードを収集・合算
   → コード昇順でソート
   → グループ内「先頭」シートの 45 行目以降に R=コード, S=内容, T=単位数, U=回数 を出力

3. シート「集計」が存在する場合
   → 各グループの全シートの V44 を合計
   → 集計シート A 列で受給者番号一致行を検索
   → 一致行の K 列に合計値を出力

4. 完了メッセージ表示
```

---

## 9. エラー時

- エラー発生時は `ErrHandler` で捕捉し、`MsgBox` で「エラーが発生しました: [説明]」を表示
- シート「集計」の取得時のみ `On Error Resume Next` で存在しない場合を無視
