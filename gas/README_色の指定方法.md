# シフト照合GAS：対象範囲の「色」指定方法

行の軸は「何行目〜何行目」が固定でないため、**行を背景色でマークし、その色で範囲を判定**する仕様にしています。  
**実行のたびにシートのマーカー列を走査する**ため、行の増減で位置が変わっても、その時点の色に合わせて正しく範囲を判定します。

---

## 1. 対象範囲の考え方

- **列**：固定（D–H, J–N, P–T, V–Z, AB–AF, AH–AL, AN–AR の7ブロック）
- **行**：「対象色1」〜「対象色6」の**6種類の色**で区切られた**5つのブロック**で判定します。

各ブロックの並びは次のとおりです。

| 順番 | 意味 | 行の範囲 |
|------|------|----------|
| 1 | 対象色1の**1行目** | マーカー行（区切り線の役割） |
| 2 | **シフト表** | 「対象色1」が付いた2本のマーカー行**の間**の行 |
| 3 | 対象色1の**2行目** | マーカー行 |
| 4 | **シフト希望** | 「対象色1の2行目」〜「対象色2の1行目」**の間**の行 |
| 5 | 対象色2の1行目 | マーカー行 |
| … | 以下、対象色2〜対象色5まで同様に繰り返し | … |
| 最後 | 対象色6の**1行目のみ** | マーカー行（5ブロック目のシフト希望は「色5の2行目」〜「色6の1行目」の間） |

つまり、

- **対象色1〜5** → 同じ色のマーカーが**2行ずつ**ある（その間がシフト表）
- **対象色6** → マーカーは**1行のみ**（最後のシフト希望の終端として使用）

という前提で、「シフト表」と「シフト希望」の範囲を自動で切り出しています。

---

## 2. 色の指定方法（2ステップ）

### ステップ1：シート上でマーカー行に色を塗る

1. 対象シートで、**「ここがブロックの区切り」**にしたい行を決めます。
2. その行の**D列**（マーカー列。別の列にしたい場合は CONFIG.MARKER_COLUMN で変更可）のセルを選択し、**塗りつぶしの色**を設定します。
   - 1ブロック目：**対象色1**（例：薄い黄）
   - 2ブロック目：**対象色2**（例：薄いオレンジ）
   - 3ブロック目：**対象色3**（例：薄い青）
   - 4ブロック目：**対象色4**
   - 5ブロック目：**対象色5**
   - 6ブロック目：**対象色6**
3. **対象色1〜5**は、同じ色のマーカーを**2行ずつ**付けてください（シフト表の「上」と「下」）。  
   **対象色6**は**1行だけ**塗ってください（最後のブロックの終端用）。
4. 行の増減で位置は変わってもかまいません。実行のたびにシートの色を確認するため、その時点のマーカーで範囲が決まります。

色の判定は**D列（4列目）の背景だけ**を見ます。  
他の列だけ塗っても認識されないので、**必ずマーカー列（デフォルトはD列）を塗る**ようにしてください。

---

### ステップ2：スクリプトに「色の16進コード」を書く

GASでは背景色を **#RRGGBB** の形式（16進コード）で扱います。  
シートで使った色と、スクリプトの `CONFIG` を一致させる必要があります。

#### 方法A：メニューで色コードを確認する（推奨）

1. スクリプトを保存し、シートを再読み込みするかメニュー「シフト照合」を出します。
2. シートで、**マーカーに使った色が付いたセル（D列）を1つ選択**します。
3. メニュー「シフト照合」→ **「選択セルの色コードを表示」** を実行します。
4. 表示されたコード（例：`#ffff00`）をコピーし、`ShiftValidation.gs` の `CONFIG` の該当色に貼り付けます。

```javascript
// 例：対象色1 が #ffff00 だった場合
var CONFIG = {
  COLOR_1: '#ffff00',  // ← ここに貼り付け
  COLOR_2: '#ffcc99',
  // ...
};
```

対象色2〜6も、それぞれその色のセルを選択して「選択セルの色コードを表示」で確認し、`COLOR_2` 〜 `COLOR_6` に設定してください。

#### 方法B：スプレッドシートの色パレットと対応表を使う

スプレッドシートの「塗りつぶしの色」で選べる色の例です（環境により微妙に違う場合があります）。

| 色の例 | 16進コード（目安） |
|--------|---------------------|
| 薄い黄 | `#ffff00` または `#fff2cc` |
| 薄いオレンジ | `#ffcc99` または `#f4b183` |
| 薄い青 | `#99ccff` または `#b4d7ee` |
| 薄い緑 | `#99ff99` または `#c6efce` |
| 薄い紫 | `#ff99ff` または `#e4c6f5` |
| 薄いグレー | `#cccccc` または `#d9d9d9` |

実際の値は**方法A**で確認するのが確実です。

---

## 3. マーカーを付ける列を変えたい場合

デフォルトは **D列（4列目）** でマーカー色を判定しています。  
別の列で色を塗りたい場合は、`CONFIG.MARKER_COLUMN` を変更してください。

```javascript
// 例：A列でマーカーを判定する場合
MARKER_COLUMN: 1
```

- 1 = A列  
- 2 = B列  
- 3 = C列  
- 4 = D列（デフォルト）  
- …

---

## 4. よくあるトラブル

- **「マーカー行が検出できません」**
  - マーカーは**D列（または CONFIG.MARKER_COLUMN で指定した列）**に塗っていますか？
  - `CONFIG` の `COLOR_1` 〜 `COLOR_6` が、シートで使った色の16進コードと一致していますか？
  - 「選択セルの色コードを表示」で、実際のコードを確認してから `CONFIG` を書き直してください。

- **「ブロックを特定できません」**
  - **対象色1〜5**は、同じ色のマーカーが**2行ずつ**ありますか？
  - **対象色6**は**1行だけ**で問題ありません。
  - 並びは「色1・色1・色2・色2・色3・色3・色4・色4・色5・色5・色6」の順になっていますか？

- **範囲が意図とずれる**
  - シフト表は「同じ色の2本のマーカー**の間**」、シフト希望は「その2本目〜次の色の1本目**の間**」です。  
    マーカーを1行ずらすと範囲が変わるので、色の付いた行の位置を確認してください。

---

## 5. まとめ

1. **行の対象範囲** ＝ D列（または指定列）の**背景色**で判定する。  
2. **色の指定** ＝ シートで塗った色の**16進コード**を、メニュー「選択セルの色コードを表示」で確認し、`CONFIG` に書く。  
3. **ブロックの並び** ＝ 対象色1〜5は「同じ色のマーカー2行」、対象色6は「1行のみ」。その並びでシフト表・シフト希望の範囲が決まる。
4. **毎回の実行** ＝ 行の増減で位置が変わっても、実行のたびにシートの色を確認するため、その時点のマーカーで正しく範囲を判定する。

この4点を守れば、行番号が変わっても「色」で正しく対象範囲を指定できます。
