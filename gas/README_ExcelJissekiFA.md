# Excel実績 F/A 入力マクロ（GAS）

選択したセル範囲に対し、「2つ左」のセルが**水色**のときだけ、半角 **F** または **A** を自動で入れます。

---

## 仕様

| 条件 | 入力する値 |
|------|------------|
| 2つ左のセルが**水色** かつ そのセルの値が「ある言葉」リストのいずれかと**部分一致** | 半角 **F** |
| 2つ左のセルが**水色** かつ **Fの対象ではない**（空・リストにない等） | **A** |
| 2つ左のセルが水色でない | 選択セルを**空**にする |

- **「ある言葉」リスト**：シート「**Excel実績自動化フラグ**」の **E列 3行目以降** に記載された文字列（空白行は無視）
- **2つ左**：選択したセルと同じ行で、列が2つ左のセル（例：G列を選択 → 参照はE列）

---

## 「A」が表示されるルール（テキスト）

**目的コード反映**で、選択したセルに **「A」** が入るのは、次のとおりです。

- **参照セル（2つ左）が水色**であり、かつ **Fの対象ではない**とき。
- 「Fの対象」＝参照セルの値が「Excel実績自動化フラグ」シート E列3行目以降のいずれかと**部分一致**する場合。それ以外はすべてAの対象。

つまり：
- 参照セルが水色で、かつ「ある言葉」リストのいずれかと部分一致 → **F**
- 参照セルが水色で、上記以外（空・リストにない・一致しない等） → **A**
- 参照セルが水色でない → 選択セルは**空**

**まとめ（一文で）**  
「Fの対象ではない、水色の参照セルすべてに対して、選択したセルに A が入力される。」

## セットアップ

1. スプレッドシートを開く
2. **拡張機能** → **Apps Script** を開く
3. このフォルダの **`ExcelJissekiFAInput.gs`** と **`ExcelJisseki12Input.gs`** の内容を、プロジェクトに追加する（請求フラグを使う場合は **`ExcelJissekiBillingFlag.gs`** も追加する）
4. **既に他のスクリプトで onOpen を使っている場合**：その onOpen の中に **`addExcelAutomationMenu();`** の1行を追加する  
   → これで「Excel自動化」メニューが既存のメニューと一緒に表示されます
5. 保存し、スプレッドシートを再読み込みする
6. メニューに **「Excel自動化」**（目的コード反映・2人付反映・請求フラグ・選択セルの色コードを表示）が表示される

---

## 「請求フラグ」がメニューに表示されない場合

1. **onOpen() で addExcelAutomationMenu() を呼んでいるか確認**  
   メニューを手で作っている（`createMenu('Excel自動化').addItem(...)` を何度も書いている）場合はやめ、**必ず `addExcelAutomationMenu();` を1回だけ呼ぶ**ようにしてください。こうすると、`ExcelJissekiFAInput.gs` の定義がそのままメニューになるため、「請求フラグ」も含まれます。

2. **Apps Script のコードが最新か確認**  
   Googleの「拡張機能」→「Apps Script」で、`ExcelJissekiFAInput.gs` に  
   `.addItem('請求フラグ', 'applyBillingFlagToSelection')`  
   の行があるか確認し、なければこのリポジトリの内容で上書きして保存してください。あわせて `ExcelJissekiBillingFlag.gs` もプロジェクトに追加してください。

3. **メニューを再読み込み**  
   スプレッドシートを**一度閉じて開き直す**と、onOpen() が実行され、メニューが作り直されます。

## 水色の設定（重要）

GASでは背景色を **#RRGGBB** の16進コードで判定します。

1. シート上で、**実際に「水色」として使っているセルを1つ選択**する
2. メニュー「Excel自動化」→ **「選択セルの色コードを表示」** を実行する
3. 表示されたコード（例：`#b4d7ee`）をコピーする
4. スクリプトの **`CONFIG.LIGHT_BLUE_HEX`** にその値を貼り付けて保存する

```javascript
var CONFIG = {
  LIGHT_BLUE_HEX: '#b4d7ee',  // ← ここを実際の水色のコードに
  WORDS_SHEET_NAME: 'Excel実績自動化フラグ',
  WORDS_COLUMN: 5,
  WORDS_FIRST_ROW: 3
};
```

---

## 使い方

1. **F** または **A** を入れたいセル範囲を選択する（例：G3:G100）
   - 選択の左端は **C列より右** にすること（2つ左のセルを参照するため）
2. メニュー「Excel自動化」→ **「目的コード反映」** を実行する
3. 処理結果の件数が表示される

---

## 注意事項

- 参照セル（2つ左）が水色でない場合は、選択セルを**空**にします
- 「ある言葉」は**部分一致**で判定します（参照セルの値に、リストのいずれかの文字列が含まれていれば F）
- リストはシート「Excel実績自動化フラグ」の E列 の**表示値**で比較します
- 水色の参照セルは、Fに該当しなければすべてAになります（空でも何か入っていても）
