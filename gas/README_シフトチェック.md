# シフト確認システム 使い方ガイド

シフト表とヘルパーのシフト希望を自動で照合し、不一致を赤字・太字でお知らせするシステムです。

---

## できること

シフト表とシフト希望を比べて、以下の3つの問題を検出します。

| # | 検出内容 | 意味 |
|---|---------|------|
| 1 | シフト表にいるがシフト希望にいない | シフト希望を出していないヘルパーが割り当てられている |
| 2 | シフト希望の時間外で支援が入っている | ヘルパーの希望時間と合っていない |
| 3 | シフト希望にいるがシフト表にいない | 希望を出しているのにシフトに入っていない |

問題のあるヘルパー名が **赤字・太字** になります。問題がなければ黒字・通常のままです。

---

## 事前準備

### 「シフトチェック」シートの設定

スプレッドシート内に「シフトチェック」というシートが必要です。以下のセルを使用します。

| セル | 内容 |
|------|------|
| **C4** | 1週目のシート名（例: `4月1週`） |
| **C5** | 2週目のシート名 |
| **C6** | 3週目のシート名 |
| **C7** | 4週目のシート名 |
| **C8** | 5週目のシート名 |
| **C9** | 6週目のシート名 |
| **D4〜D9** | 「確認」と入力するとチェック実行 |
| **E4〜E9** | 「黒」と入力するとリセット実行 |

### 各週シートの構成

各週のシートは以下のルールで作られている必要があります。

- **D列** に背景色 `#073763`（濃い青）が **2箇所** ある
  - **1箇所目〜2箇所目の間** → シフト表（実際の支援予定）
  - **2箇所目の下** → シフト希望（ヘルパーの希望）
- 1シートに **最大7日分** のシフトが横に並んでいる

#### 各日の列構成

| 日目 | ヘルパー名 | 開始時刻 | 終了時刻 |
|------|-----------|---------|---------|
| 1日目 | D列 | F列 | G列 |
| 2日目 | AA列 | AC列 | AD列 |
| 3日目 | AX列 | AZ列 | BA列 |
| 4日目 | BU列 | BW列 | BX列 |
| 5日目 | CR列 | CT列 | CU列 |
| 6日目 | DO列 | DQ列 | DR列 |
| 7日目 | EL列 | EN列 | EO列 |

※ 2日目以降は前の日から23列ずれた位置です。

## 使い方

### 方法1: セル入力で実行（スマホでも使えます）

#### チェックを実行する

1. 「シフトチェック」シートを開く
2. C列（C4〜C9）にチェックしたい週のシート名が入っていることを確認
3. 対応する **D列のセル** に `確認` と入力して Enter

| 入力するセル | 動作 |
|-------------|------|
| D4 に `確認` | → C4 のシート名で1週目をチェック |
| D5 に `確認` | → C5 のシート名で2週目をチェック |
| D6 に `確認` | → C6 のシート名で3週目をチェック |
| D7 に `確認` | → C7 のシート名で4週目をチェック |
| D8 に `確認` | → C8 のシート名で5週目をチェック |
| D9 に `確認` | → C9 のシート名で6週目をチェック |

チェック完了後、セルに `完了(エラー○件)` と表示されます。

#### リセットする（赤字を黒に戻す）

対応する **E列のセル** に `黒` と入力して Enter

| 入力するセル | 動作 |
|-------------|------|
| E4 に `黒` | → 1週目のシートをリセット |
| E5 に `黒` | → 2週目のシートをリセット |
| E6〜E9 | → 同様に対応する週をリセット |

リセット完了後、セルに `リセット完了` と表示されます。

---

### 方法2: メニューから実行（PCのみ）

画面上部のメニューバーに「**シフト確認**」メニューがあります。

| メニュー項目 | 動作 |
|-------------|------|
| 1週目チェック〜6週目チェック | 各週を個別にチェック |
| 全週チェック | C4〜C9 に入っている全シートを一括チェック |
| 1週目リセット〜6週目リセット | 各週を個別にリセット |
| 全週リセット | 全シートを一括リセット |

---

## チェック結果の見方

チェック実行後、各週シートのヘルパー名セルが以下のように変わります。

| 表示 | 意味 |
|------|------|
| **赤字・太字** | 問題あり（上記3種類のいずれかに該当） |
| 黒字・通常 | 問題なし |

### 具体例

- 「田中」さんがシフト表に入っているのに、シフト希望を出していない
  → **シフト表側** の「田中」が赤字・太字になる

- 「佐藤」さんが希望 9:00〜17:00 なのに、18:00〜19:00 の支援が入っている
  → **シフト表側** のその行の「佐藤」が赤字・太字になる

- 「鈴木」さんがシフト希望を出しているのに、シフト表に入っていない
  → **シフト希望側** の「鈴木」が赤字・太字になる

## よくある質問

### Q. チェック後に修正したら、もう一度チェックできますか？

はい。修正後にもう一度 D列に `確認` と入力するか、メニューからチェックを実行してください。前回の赤字は自動的に黒に戻り、最新の状態で再チェックされます。

### Q. エラーが0件なのに赤字が残っています

前回のチェック結果が残っている可能性があります。E列に `黒` と入力するか、メニューからリセットを実行してください。

### Q. 「シート名が入力されていません」と出ます

「シフトチェック」シートの C列（C4〜C9）に対象のシート名が入っていません。シート名を入力してから再度実行してください。

### Q. 「シートが見つかりません」と出ます

C列に入力したシート名と、実際のシート名（タブの名前）が一致していません。スペースや全角・半角の違いがないか確認してください。

### Q. 「マーカー行が見つかりません」と出ます

対象シートの D列に背景色 `#073763`（濃い青）が2箇所ないと動作しません。シートの構成を確認してください。

### Q. スマホからも使えますか？

はい。「シフトチェック」シートの D列・E列にセル入力する方法であればスマホからも使えます。メニューはPC版のみです。
