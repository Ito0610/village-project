# Excel実績 請求フラグ入力（GAS）

選択したセル範囲に対し、「**7つ左**」のセルの背景色を参照して「居宅」または「大田区」を自動で入れます。

---

## 仕様

| 条件 | 入力する値 |
|------|------------|
| 7つ左のセルが **#ff9900（オレンジ）** | **居宅** |
| 7つ左のセルが **#00ffff（水色）** かつ そのセルの値が「Excel実績自動化フラグ」**I列3行目以降**のいずれかと**完全一致** | **大田区** |
| 上記以外 | 選択セルを**空**にする |

- **7つ左**：選択したセルと同じ行で、列が7つ左のセル（例：O列を選択 → 参照はH列）
- メニュー「Excel自動化」は **ExcelJissekiFAInput.gs** の `addExcelAutomationMenu()` で追加されます。本ファイルを同じプロジェクトに追加すると「請求フラグ」がメニューに表示されます。

---

## セットアップ

1. スプレッドシートの **拡張機能** → **Apps Script** を開く
2. **ExcelJissekiFAInput.gs** で「Excel自動化」メニューが既に追加されている前提で、本ファイル **`ExcelJissekiBillingFlag.gs`** を同じプロジェクトに追加する
3. 保存し、スプレッドシートを再読み込みする
4. メニュー「Excel自動化」→ **「請求フラグ」** が利用可能になる

---

## 使い方

1. 「居宅」または「大田区」を入れたいセル範囲を選択する（例：O3:O100）
   - 選択の左端は **H列より右** にすること（7つ左のセルを参照するため）
2. メニュー「Excel自動化」→ **「請求フラグ」** を実行する
3. 処理結果の件数（居宅・大田区・対象外）が表示される

---

## 色の変更

スクリプト内の **`BILLING_CONFIG`** で色コードを変更できます。

- `ORANGE_HEX`: オレンジ（居宅判定用）。既定は `#ff9900`
- `CYAN_HEX`: 水色（大田区判定用）。既定は `#00ffff`

実際のシートの色と異なる場合は、メニュー「Excel自動化」→「選択セルの色コードを表示」でセルを選択してコードを確認し、`ExcelJissekiBillingFlag.gs` の `BILLING_CONFIG` に反映してください。

---

## 注意事項

- シート「**Excel実績自動化フラグ**」が存在しないとエラーになります
- 大田区の判定は、参照セルの**表示値**が「Excel実績自動化フラグ」の **I列3行目以降**のいずれかと**完全一致**するかで行います
