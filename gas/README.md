# シフト表・シフト希望 照合チェック（GAS）

Googleスプレッドシート上で、シフト表とシフト希望を照合し、次の3種類の間違いを検知して**赤色・太字**でマークします。

1. **シフト表にだけある名前** … シフト希望に記載のないヘルパーがシフト表に入っている → シフト表のヘルパー名を赤・太字
2. **シフト希望時間外の支援** … 支援時間がシフト希望のいずれの時間帯にも含まれない → シフト表のヘルパー名を赤・太字
3. **シフト希望にだけある名前** … シフト希望にはあるがシフト表に1件もない → シフト希望のヘルパー名を赤・太字

---

## セットアップ

1. スプレッドシートを開く
2. **拡張機能** → **Apps Script** を開く
3. デフォルトの `Code.gs` を、このフォルダの **`ShiftValidation.gs`** の内容で置き換える（または新規ファイルとして追加）
4. 保存後、対象シートで**行の区切り**にしたい行の **D列** を、説明どおり「対象色1〜6」で塗る
5. **色の16進コード**をスクリプトの `CONFIG` に合わせる（メニュー「選択セルの色コードを表示」で確認可能）
6. スプレッドシートを再読み込みすると、メニューに **「シフト照合」** が表示される

詳細は **`README_色の指定方法.md`** を参照してください。

---

## スマホ・セル入力で実行（シフトチェックシート）

「シフトチェック」シートの **B2** に「日曜」「月曜」…「土曜」または「黒」を入力すると、編集確定で照合が実行されます。メニューが出ないスマホでも利用できます。

- **照合対象シート名**：毎月シート名が変わる場合は、**G3 セル**に照合したいシート名（例: `3月`）を入力してください。実行時にこのセルを参照して、そのシートに対して照合します。スクリプトの編集は不要です。
- 参照するセルを変えたい場合は、スクリプト内 `EDIT_TRIGGER.DATA_SHEET_NAME_CELL`（例: `[3, 7]` = G3）を変更してください。

---

## 使い方

- **日曜チェック** … D〜H列のみ照合（実行時間を分散し、タイムアウトを防ぐ）
- **月曜チェック** … J〜N列のみ照合
- **火曜チェック** … P〜T列のみ照合
- **水曜チェック** … V〜Z列のみ照合
- **木曜チェック** … AB〜AF列のみ照合
- **金曜チェック** … AH〜AL列のみ照合
- **土曜チェック** … AN〜AR列のみ照合
- **選択セルの色コードを表示** … 色設定用。マーカーに使ったセルを選択して実行すると、その背景色の16進コードが表示される

---

## 対象範囲

- **列**：D–H, J–N, P–T, V–Z, AB–AF, AH–AL, AN–AR の7ブロック（各5列：ヘルパー名・利用者名・開始・終了・備考）
- **行**：D列（または設定したマーカー列）の背景色で区切られた「シフト表」「シフト希望」のブロック（最大5ブロック × 7列 = 35エリア）。対象色1〜5は同じ色2行、対象色6は1行のみ。実行のたびにシートの色を確認するため、行の増減で位置が変わっても対応できる。ブロックが存在しない列・行は自動的にスキップされる
