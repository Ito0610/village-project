# シフト照合チェック 使い方ガイド（社内用）

シフト表とシフト希望を照合し、**間違いを赤・太字で表示**するツールです。

---

## このツールで検出する3つの間違い

| 種類 | 内容 | マークされる場所 |
|------|------|------------------|
| 1 | シフト表にだけある名前 | シフト希望にいないのにシフト表に入っている → **シフト表**のヘルパー名が赤・太字 |
| 2 | シフト希望時間外の支援 | 支援時間がシフト希望のどの時間帯にも入っていない → **シフト表**のヘルパー名が赤・太字 |
| 3 | シフト希望にだけある名前 | シフト希望にはあるがシフト表に1件もない → **シフト希望**のヘルパー名が赤・太字 |

赤・太字になった箇所を確認し、表の修正や希望の確認をしてください。

---

## 使い方（2通り）

### 方法A：パソコンでメニューから実行

1. 照合したい**シート**（例：3月のシフト）を開く
2. 画面上部メニュー **「シフト照合」** をクリック
3. チェックしたい曜日を選ぶ  
   - **日曜チェック** … 日曜の列だけ照合  
   - **月曜チェック** … 月曜の列だけ照合  
   - **火曜〜土曜チェック** … 同様に該当曜日だけ照合  
4. 実行後、該当範囲内で間違いがあれば**赤・太字**で表示されます

**ポイント**  
- 曜日ごとに分けて実行するため、タイムアウトしにくく、スマホでも負荷が軽くなります。  
- 全部まとめてではなく、**日曜→月曜→…** と順に実行してください。

---

### 方法B：スマホ・セル入力で実行（シフトチェックシート）

メニューが出ないスマホや、セル操作だけで実行したい場合に使います。

1. **「シフトチェック」** という名前のシートを開く
2. **G3** に、照合したいシート名を入力する（例：`3月`）  
   - 毎月シート名が変わる場合は、ここを変えるだけでOKです
3. **B2** に次のいずれかを入力して**確定**（Enter）する  
   - **日** または **日曜** … 日曜だけ照合  
   - **月** または **月曜** … 月曜だけ照合  
   - **火** 〜 **土** または **火曜** 〜 **土曜** … 該当曜日だけ照合  
   - **黒** または **リセット** … 赤・太字をいったん黒・通常に戻す
4. 確定すると自動で照合が実行され、B2に「実行しました（〇曜チェック）」などと表示されます

**注意**  
- G3 が空のまま B2 に曜日を入力すると、「照合対象のシート名を入力してください」と表示されます。その場合は G3 にシート名（例：3月）を入れてから再度 B2 で実行してください。

---

## 赤・太字を解除したいとき

- **パソコン**：メニュー「シフト照合」→ **「名前を黒色に戻す」** を実行
- **スマホ（シフトチェックシート）**：**B2** に **「黒」** または **「リセット」** と入力して確定

※「名前を黒色に戻す」は、ヘルパー名の列（D, J, P など）の文字色・太字だけを黒・通常に戻します。表の他の列は変更しません。

---

## 照合の対象になる範囲

- **列**：日曜〜土曜の7ブロック（各ブロック5列：ヘルパー名・利用者名・開始・終了・備考）
- **行**：シートの **D列** に付けた**色**で「シフト表」「シフト希望」のブロックが判定されます。  
  （色の設定は初回セットアップで行います。詳しくは `README.md` と `README_色の指定方法.md` を参照）

---

## よくあること

- **「照合用シートが見つかりません」**  
  → G3（または指定しているセル）のシート名が、実際のシート名と一致しているか確認してください。スペースや全角・半角の違いに注意。
- **スマホでB2に入力しても反応しない**  
  → シート名が「シフトチェック」になっているか、G3 に照合対象シート名が入っているか確認してください。編集トリガーが有効になっているかも確認してください。
- **色で範囲が分かれていない**  
  → 初回セットアップで、D列に「対象色1〜6」を正しく塗り、スクリプトの CONFIG の色コードと合わせてください。`README_色の指定方法.md` を参照。

---

## まとめ

| やりたいこと | 操作 |
|--------------|------|
| パソコンで曜日ごとに照合 | 対象シートを開く → メニュー「シフト照合」→ 該当曜日のチェック |
| スマホで照合 | 「シフトチェック」シートで G3 にシート名、B2 に「日」「月」…「土」のいずれかを入力して確定 |
| 赤・太字を戻す | メニュー「名前を黒色に戻す」、または B2 に「黒」と入力して確定 |

不明点や不具合があれば、担当者に連絡してください。
