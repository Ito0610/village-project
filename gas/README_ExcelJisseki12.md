# Excel実績 1/2 入力マクロ（GAS）

選択したセル範囲に対し、「**6つ左**」のセルが**水色**のときだけ、半角 **1** または **2** を自動で入れます。  
利用者様名が「Excel実績自動化フラグ」のA列と一致する場合は「2」を入れ、さらに「7つ左」のヘルパー名が**2人以上**のときだけ「2」、**1人**のときは「1」にします。

---

## 仕様

| 条件 | 入力する値 |
|------|------------|
| 6つ左のセルが**水色でない** | 何もしない（既存の値はそのまま） |
| 6つ左が水色 かつ 利用者様名がリストに**一致しない** | 半角 **1** |
| 6つ左が水色 かつ 利用者様名がリストに**一致** かつ 7つ左のヘルパーが**1人** | 半角 **1** |
| 6つ左が水色 かつ 利用者様名がリストに**一致** かつ 7つ左のヘルパーが**2人以上** | 半角 **2** |

- **利用者様名リスト**：シート「**Excel実績自動化フラグ**」の **A列 3行目以降**（空白行は無視）
- **6つ左**：選択したセルと同じ行で、列が6つ左のセル（利用者様名）
- **7つ左**：同じ行で列が7つ左のセル（ヘルパー名。スペースで区切られた人数で 1/2 を判定）
- **ヘルパー人数**：7つ左のセルを**全角・半角スペース**で区切ったときの名前の個数（例：`市川　奥原(初)　荻原` → 3人 → 「2」）

---

## セットアップ

1. スプレッドシートを開く
2. **拡張機能** → **Apps Script** を開く
3. **ExcelJisseki12Input.gs** の内容を新規ファイルとして追加
4. メニューを出すため、次のどちらかを行う
   - **1/2入力だけ使う場合**：`ExcelJisseki12Input.gs` 内の `onOpen12` を **`onOpen`** にリネームして保存
   - **F/A入力と両方使う場合**：`ExcelJissekiFAInput.gs` の `onOpen` に、次の3行を追加して保存  
     （`addToUi();` の直前に追加）
     ```javascript
     ui.createMenu('1/2入力')
       .addItem('選択範囲に1/2を反映', 'apply12ToSelection')
       .addItem('選択セルの色コードを表示', 'showSelectedCellColor12')
       .addToUi();
     ```
5. スプレッドシートを再読み込みする
6. メニューに **「1/2入力」** が表示される

---

## 水色の設定（重要）

GASでは背景色を **#RRGGBB** の16進コードで判定します。F/A入力と同じ水色を使う場合は、同じコードを **CONFIG_12.LIGHT_BLUE_HEX** に設定してください。

1. シート上で、**実際に「水色」として使っているセルを1つ選択**する
2. メニュー「1/2入力」→ **「選択セルの色コードを表示」** を実行する
3. 表示されたコード（例：`#b4d7ee`）を、スクリプトの **CONFIG_12.LIGHT_BLUE_HEX** に設定して保存する

---

## 使い方

1. **1** または **2** を入れたいセル範囲を選択する（例：H3:H100）
   - 選択の左端は **H列（8列目）より右**にすること（6つ左・7つ左を参照するため）
2. メニュー「1/2入力」→ **「選択範囲に1/2を反映」** を実行する
3. 処理結果の件数が表示される

---

## 注意事項

- 6つ左のセルが水色でない行は、**一切書き換えません**
- 利用者様名は、シート「Excel実績自動化フラグ」の A列 を**表示されている値**で比較します（数式の結果で判定）
- ヘルパー名は**全角スペース・半角スペース**の両方で区切って人数を数えます
