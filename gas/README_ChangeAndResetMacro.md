# 変更＆リセットマクロ（Excel VBA）

## 概要
`ChangeAndResetMacro.bas` は、月初の一括「変更＆リセット」を行うExcel用VBAマクロです。

## 実行時の確認
マクロ実行時に次のメッセージが表示されます。

- **「変更＆リセットを行っていいですか？」**  
  「はい」で実行、「いいえ」で中止します。

## 処理内容

### 処理1（月初変更＆リセットシートからコピー）
- **参照シート**: 「月初変更＆リセット」
- **参照セル**: C2 → 対象シートの **D3** にコピー  
- **参照セル**: C3 → 対象シートの **F3** にコピー  
- **対象シート**: シート名に「様」が含まれるシート

### 処理2（指定範囲の値削除）
- **対象シート**: シート名に「様」が含まれるシート
- **削除範囲**: 次の列の **16〜35行目** の**値**を削除  
  - A列, C列, F列, G列, H列, J列, K列, M列, P列  
  - R列, S列, T列, U列, V列, W列  
- 書式は残し、セルの値のみ削除します。  
- **結合セル対応**: 結合されたセルも値のみクリアします（結合・書式は維持）。

## 使い方
1. ExcelでVBE（Alt+F11）を開く
2. メニュー「ファイル」→「ファイルのインポート」で `ChangeAndResetMacro.bas` を読み込む  
   または、標準モジュールを追加し、`ChangeAndResetMacro.bas` の内容を貼り付ける
3. ブックに「月初変更＆リセット」シートがあり、C2・C3に必要な値が入っていることを確認
4. `ChangeAndReset` マクロを実行（F5 または「マクロ」ダイアログから実行）

## 注意
- シート「月初変更＆リセット」が存在しない場合はエラーメッセージを表示して終了します。
- 対象シートは「様」を含む名前のシートのみです（例: 「山田太郎様」「佐藤花子様」など）。
